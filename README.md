# Encajonado - Proyecto 2D Puzzle (DAM)

# Encajonado es un videojuego de puzzles desarrollado en Unity para la asignatura de Proyecto Intermodular de 2º de DAM. El objetivo es desplazar cajas hacia sus metas optimizando el número de pasos.

# 

# NOTA IMPORTANTE: Seguridad y Ejecución

# Por motivos de seguridad y siguiendo las directrices de la entrega, este repositorio no incluye los archivos de configuración de Firebase (google-services.json).

# 

# Debido a esto, la escena de Login/Registro no podrá validar credenciales ni conectar con la base de datos en tiempo real.

# 

# Cómo visualizar el avance del proyecto:

# Para revisar las mecánicas de juego, la generación de niveles y la interfaz, por favor siga estos pasos dentro del editor de Unity:

# 

# Vaya a la carpeta Assets/Scenes/.

# 

# Abra la escena llamada MundoJuego.

# 

# Pulse el botón Play.

# 

# Desde aquí podrá probar el movimiento, el sistema de empuje de cajas y el contador de pasos sin necesidad de autenticación.

# 

# Funcionalidades Implementadas:

# 

# Sistema de Movimiento: Con "WASD"

# 

# Mecánica de Reinicio: Con "R"
